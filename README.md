# conjugate-positive-numbers

This project is in golang.
To test, you must install it. 

Follow this link if any doubts:
https://go.dev/doc/install

## Build and Test application

To build:

```
go build
```

To test, you can pass any collection with integer by slash, eg:

```
./conjugate-positive-numbers 6-4-2
```


## Notes

* I did not implement any validation or unit tests, but those are essential if we 
gonna build something for production environment
